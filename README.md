# Badu AI — AI Automation Agency Website

A professional agency website with 3 live AI demo tools, built with pure HTML/CSS/JS. No frameworks, no build step. Deploy instantly to GitHub Pages.

## 🚀 Live Demos

| Demo | Description |
|------|-------------|
| `index.html` | Agency landing page |
| `demo-content.html` | AI Content Agency — generate blogs, social posts, emails |
| `demo-chatbot.html` | Chatbot Builder — configure & chat with an AI bot |
| `demo-leadgen.html` | Lead Gen & Outreach — personalised cold email generator |

## ⚡ Deploy to GitHub Pages (Free)

1. Create a new GitHub repo (e.g. `neuralcraft`)
2. Upload all 4 HTML files
3. Go to **Settings → Pages → Source: main branch / root**
4. Your site is live at `https://yourusername.github.io/neuralcraft`

## 🔑 Activating Live Claude AI

Each demo has an optional **Anthropic API Key** field. Without it, smart sample responses are shown (great for demos). With it, real Claude AI powers the outputs.

Get your API key at: https://console.anthropic.com

## 🛠️ Customisation

Edit these in each file:
- Business name: search `NeuralCraft` and replace
- Contact email: search `hello@neuralcraft.ai`  
- Location: search `Rajshahi, Bangladesh`
- Colors: edit `:root` CSS variables at top of each file
